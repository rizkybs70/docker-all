Siapkan DNSCRYPT-PROXY.TOML , DOCKERFILE.TXT (ada di folder bahan)

simpan di home/username ente klo login pake root ya di /root/

Sesuaikan isi dari DNSCRYPT-PROXY.TOML , DOCKERFILE.TXT nya 0.0.0.0:5300

buka command di folder tsb ketik : 
$ sudo docker build -t dnscrypt -f dockerFile.txt .

buka portainer docker

buat network bridge

add container dengan images dnscyp  pake network tadi yang di buat dengan ip 172.20.0.4

ketik command : nslookup detik.com 172.20.0.4 kalo konek tinggal gabungin di pihole

pihole yg sudah terinstal masukin jg ke network tadi (bridge) dengan ip 172.20.0.2
buka pihole > setting> dns > centang custom 1 dan masukin 172.20.0.4#5300
